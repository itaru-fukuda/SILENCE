# Skill: Imaging Physics

## 概要
光学カメラおよび SAR (合成開口レーダー) の物理的特性を、WebGL / GLSL シェーダーを用いて再現するための専門スキル。

## 光学シミュレーション (Cox-Munk モデル)
- **サングリント（海面反射）**
  - 風速 $U$ に基づく海面反射を計算する GLSL 関数を実装すること。
  - 二変量ガウス分布による傾斜確率密度を用い、正確な反射モデルを構築する。
  - 昼間の海面で発生する白飛び（露光オーバー）現象を再現できるように物理ベースのレンダリング (PBR) または専用シェーダを構築する。

## SAR (疑似レンダリング)
- **頂点シェーダー (Vertex Shader)**
  - レーダーの照射距離（スラントレンジ）に基づく **レイオーバー**（距離による頂点シフト・倒れ込み現象）を実装すること。
- **フラグメントシェーダー (Fragment Shader)**
  - SAR画像特有の **乗法的なスペックルノイズ** を実装すること。
  - 指数分布 $X = - \ln(U)$ （$U$ は $(0, 1]$ の一様乱数）に基づくノイズを生成し、ピクセル値に乗算して反射強度をシミュレートする。
